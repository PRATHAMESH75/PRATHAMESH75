## 📌 Featured Project

### 🎧 [EchoSim](https://github.com/PRATHAMESH75/EchoSim) — Sentiment Simulator

A launch-focused product workflow that lets product teams pressure-test messaging, pricing, and market risk **before shipping**. It turns a structured product brief into a knowledge graph, expands the audience into archetype-driven consumer segments, and runs **three scenarios in parallel** — baseline, competitive pressure, and crisis response — then surfaces sentiment movement, objections, faction shifts, and archetype heatmaps in a single dashboard.

**Highlights**
- 🧠 Brief → **knowledge graph** → archetype-based population synthesis (LLM + Zep)
- ⚡ **Parallel campaign execution** with async, poll-tracked preparation tasks
- 📊 Per-simulation **sentiment caching** with log-driven invalidation
- 🔒 Locked-down production defaults — required `SECRET_KEY`, debug off, origin-based CORS
- 🐳 Single-image **Docker** deploy: backend builds + serves the compiled frontend from one process

**Tech:** Python · Flask · Vue · LLM (OpenAI-compatible) · Zep knowledge graph · Docker
